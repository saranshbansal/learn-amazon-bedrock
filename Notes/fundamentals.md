# Amazon Bedrock Fundamentals

## Service Endpoints

Amazon Bedrock provides the following service endpoints:

### bedrock

Control Plane APIs:

- Managing models
- Training models
- Deploying models

### bedrock-runtime

Data Plane APIs:

- Making inference requests for models hosted in Amazon Bedrock

### bedrock-agent

Control Plane APIs:

- Creating and managing agents
- Managing knowledge bases
- Prompt management
- Prompt flows

### bedrock-agent-runtime

Data Plane APIs:

- Invoking agents and flows
- Querying knowledge bases
